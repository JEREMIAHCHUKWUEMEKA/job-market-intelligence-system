# Job Market Intelligence System

## Overview

The Job Market Intelligence System is a Python-based data analytics project that collects job postings from public job APIs, processes and analyzes the data, extracts in-demand technical skills, and generates actionable insights about the job market.

The project demonstrates key data engineering, analytics, and visualization concepts by transforming raw job listings into meaningful market intelligence.

---

## Objectives

This project aims to answer questions such as:

* What skills are currently most in demand?
* Which companies are hiring the most?
* Which locations have the highest number of openings?
* What job categories dominate the market?
* Which technologies frequently appear together in job descriptions?

---

## Features

### Data Collection

* Collects job listings from the Remotive Jobs API.
* Extracts relevant job information including:

  * Job Title
  * Company
  * Location
  * Salary
  * Job Type
  * Category
  * Publication Date
  * Description

### Data Cleaning

* Removes HTML tags from job descriptions.
* Standardizes location information.
* Handles missing values.
* Removes duplicate records.
* Parses publication dates into a consistent format.

### Skill Extraction

Automatically identifies technical skills mentioned in job descriptions, including:

* Python
* JavaScript
* TypeScript
* React
* Node.js
* AWS
* Docker
* Kubernetes
* SQL
* PostgreSQL
* MongoDB
* TensorFlow
* PyTorch
* Machine Learning
* NLP

### Analytics Engine

Generates insights such as:

* Top Skills
* Top Hiring Companies
* Top Hiring Locations
* Top Job Categories
* Technology Co-Occurrence Analysis

### Visualization

Automatically generates charts for:

* Skill Frequency Distribution
* Top Hiring Companies
* Top Hiring Locations

## Installation

### Clone the Repository

```bash
git clone https://github.com/JEREMIAHCHUKWUEMEKA/job-market-intelligence-system.git
cd job-market-intelligence-system
```

### Create a Virtual Environment

```bash
python -m venv venv
```

### Activate the Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/macOS:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Download spaCy Model

```bash
python -m spacy download en_core_web_sm
```

---

## Running the Project

```bash
python main.py
```

Generated outputs will be saved inside the `data` directory.
